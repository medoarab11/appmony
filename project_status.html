AppMony - Personal Finance App

A full-featured personal finance mobile app with multilingual and multicurrency support, offline capabilities, and premium subscription features.

Project Overview

AppMony is a comprehensive personal finance application designed to help users track expenses, manage budgets, and gain insights into their financial habits. The app supports multiple languages and currencies, works offline, and offers premium features through subscription.

Key Features:

  • Multilingual support with 10+ languages
  • Multicurrency support with all ISO 4217 codes
  • Offline-first architecture with local database
  • Expense tracking and categorization
  • Budget management and alerts
  • Financial reports and analytics
  • Premium subscription with advanced features
  • AdMob integration for free tier

Technical Stack

  • Framework: Flutter
  • State Management: Provider, Bloc
  • Database: SQLite (via sqflite)
  • Authentication: Firebase Auth
  • Cloud Storage: Firebase Firestore
  • Ads: Google AdMob
  • In-App Purchases: Google Play Billing
  • Analytics: Firebase Analytics
  • Charts: fl_chart
  • Localization: flutter_localizations
